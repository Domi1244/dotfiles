dotfiles
========

[Bork](https://github.com/mattly/bork)-based dotfile setup for Mac and Linux systems.

### Available commands

```sh
$ ./dotfiles [<files>]
# Run the assertions of a config script.
# If no files are provided, all assertions will be run.
```
